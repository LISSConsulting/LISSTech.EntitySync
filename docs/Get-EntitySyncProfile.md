# Get-EntitySyncProfile

## SYNOPSIS

Lists saved EntitySync connection profiles.

## SYNTAX

```powershell
Get-EntitySyncProfile [<CommonParameters>]
```

## DESCRIPTION

Returns profile names, the vendors stored in each profile, the default-profile marker, and the local profile-store path. Secret values are never returned.

## EXAMPLES

```powershell
Get-EntitySyncProfile
```
